# 关于git的几个基本知识点
首先 ，git是一个分布式的管理工具，他一般是具有其他管理系统所没有的优势
而且，我们可以放心的修改以及团队协作
我们如果是在本地新建一个仓库的话
就是 git init 
然后我们据新建了一个仓库
之后我们如果是在本地克隆一个仓库的话就是 git clone 
这个命令也可以直接克隆远程仓库
比如说远程服务器上的仓库
然后git的管理是通过三棵树来实现的，分别是
本地文件 index 暂存区 然后是 head
这几咳树有各自的作用，当我们在本地做了修改
首先是 git add
添加到暂存区里面去，然后呢是
git commit -m ""
提交
如果是远端服务器的话
需要git push 上去
然后是分支的概念
